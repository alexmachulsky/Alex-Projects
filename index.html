About

I am a DevOps Engineer with expertise in cloud infrastructure automation and modern deployment practices. My focus lies in building scalable, resilient systems that enable teams to deliver software efficiently and reliably.

With hands-on experience in containerization technologies like Docker and Kubernetes, I design and implement CI/CD pipelines that streamline development workflows. Passionate about bridging operations and automation to deliver production-ready systems while continuously learning and improving through real-world projects.

AI ChatBot Assistant

An intelligent AI-powered chatbot that can assist with any topic - from technology and science to business and everyday questions. Built with Flask and Ollama (local LLM), this project demonstrates modern AI integration with containerized microservices architecture. Features include real-time chat interface with glassmorphism design, conversation history, Docker Compose orchestration with Ollama server, Kubernetes manifests for production deployment, and comprehensive CI/CD pipeline with security scanning. Runs completely locally with no external API dependencies using Llama 3.1 8B model for high-quality responses.
